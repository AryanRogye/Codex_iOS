# Security Policy

## Reporting a vulnerability

Please do not open public issues for security-sensitive reports.

Email details to the maintainer and include:

1. Affected component (`relay`, `iOS app`, or both)
2. Steps to reproduce
3. Impact assessment
4. Suggested mitigation (if available)

## Scope

Reports are prioritized for:

- Authentication bypass
- Unauthorized data access
- Remote code execution paths
- Relay exposure or token leakage risks
